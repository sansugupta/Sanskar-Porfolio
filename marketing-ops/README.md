# Marketing Ops Workspace

This folder is the long-term operating memory for InfraBuild Partners marketing work.

## Why it exists
- Preserve strategy decisions across AI sessions
- Store reusable campaign ideas, automation plans, and profile copy
- Give any future AI enough context to continue marketing work without re-discovery

## What to update
- `brand-memory.md`
  - Source of truth for positioning, channel goals, audiences, offers, and campaign direction
- `session-log.md`
  - Short notes for each meaningful marketing session or decision
- `automation-playbook.md`
  - Current recommended automation stack and workflows
- `linkedin-company-profile.md`
  - Reusable company-page copy, descriptions, and section text

## Working rule
- If a future session changes pricing, campaign angles, offer focus, audience targeting, or automation tooling, update these files in the same branch.
